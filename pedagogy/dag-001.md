# DAG-001: The Causal Graph Admits a Causal Set Structure

## Intuition (no jargon)

A **causal set** is a partial order that is also locally finite: between any two events, there are only finitely many intermediate events. Think of spacetime chopped into discrete pieces, where you can always count the steps between two events. DAG-001 proves that the COG causal graph is not just acyclic (CAUS-001) but actually forms a causal set — the partial order is locally finite and the Alexandrov intervals (sets of events "between" two given events) are always finite. This is the discrete spacetime structure that underlies the entire COG program.

## Precise Statement

Let $G = (V, E)$ be the COG causal graph with vertex set $V$ (finite, `Fintype V`) and causal relation $\leq$ (the reflexive transitive closure of $E$). DAG-001 states that $(V, \leq)$ is a **causal set**:

1. **Partial order:** $\leq$ is reflexive, antisymmetric, and transitive.
2. **Local finiteness:** For all $u, v \in V$, the **Alexandrov interval**
   $$[u, v] := \{w \in V \mid u \leq w \leq v\}$$
   is finite (i.e., `Finset.card (alexandrov u v) < ∞`).

Since $V$ is assumed finite in the Lean formalization, local finiteness is automatic. The non-trivial content is that $\leq$ is a genuine partial order (antisymmetry, i.e., no directed cycles — building on CAUS-001).

## Key Proof Steps

- **Reflexivity:** $v \leq v$ holds by including the empty path from $v$ to $v$. Proved directly from the definition of reflexive-transitive closure in `CausalGraphTheory/CausalGraph.lean`.
- **Transitivity:** If $u \leq v$ and $v \leq w$ (paths $u \to^* v$ and $v \to^* w$ exist), concatenating yields $u \to^* w$. Proved by induction on path length.
- **Antisymmetry (from CAUS-001):** If $u \leq v$ and $v \leq u$ with $u \neq v$, there would be a directed cycle — contradicting `causal_graph_is_dag` (CAUS-001). Hence $u \leq v \wedge v \leq u \Rightarrow u = v$.
- **Local finiteness:** Over a finite vertex set (`Fintype V`), every subset is automatically finite. Any Alexandrov interval is a subset of $V$ and hence has `Finset.card` bounded by `Fintype.card V`.
- **Packaging:** The combined proof assembles reflexivity, antisymmetry, transitivity, and local finiteness into the `IsCausalSet` structure, with CAUS-001 (`causal_graph_is_dag`) invoked as a sub-lemma for antisymmetry.

## Implications for COG

- **Foundation for TICK-001:** The discrete tick propagation is well-defined precisely because the causal set has locally finite intervals — each tick advances exactly one step in a well-ordered chain.
- **Lorentz invariance (future work):** Causal set theory predicts Lorentz invariance to emerge statistically from a locally finite partial order. DAG-001 establishes the combinatorial foundation for this in the COG setting.
- **DIST-001 (distributivity):** The distributivity of the causal update rule over the partial order structure uses the DAG property. DAG-001 makes the partial order explicit and usable.
- **Counting paths:** The locally finite interval structure allows counting the number of causal paths between two events — a quantity that enters the mass observable definition in MASS-001.
