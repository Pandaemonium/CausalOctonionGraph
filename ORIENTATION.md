## 3. Mathematical Foundations

### 3.1 Octonions (O)

The octonion algebra O is an 8-dimensional normed division algebra with basis
{e₀, e₁, ..., e₇}, where e₀ = 1 is the real unit. Key properties:

- **Non-commutative:** e_i · e_j ≠ e_j · e_i in general
- **Non-associative:** (e_i · e_j) · e_k ≠ e_i · (e_j · e_k) in general
- **Alternative:** (e_i · e_i) · e_j = e_i · (e_i · e_j) — *two equal arguments always associate*
- **4th normed division algebra** (after ℝ, ℂ, ℍ): existence is why 3+1 dimensions

The **Alternativity Theorem** (Artin): the sub-algebra generated by any TWO
elements is always associative. Non-associativity requires all THREE elements
to be distinct and off-Fano (see §3.2).

### 3.2 The Fano Plane

The multiplication of imaginary units e₁..e₇ is encoded by the **Fano plane**
PG(2,2): 7 points, 7 lines, 3 points per line, 3 lines per point.

If {i,j,k} is a Fano line (a collinear triple), then e_i · e_j = ±e_k
(sign depends on cyclic order). The **locked** 7 directed triples (from
`rfc/CONVENTIONS.md`) are:

```
L1: (1,2,3)  → e₁·e₂ = +e₃
L2: (1,4,5)  → e₁·e₄ = +e₅
L3: (1,7,6)  → e₁·e₇ = +e₆
L4: (2,4,6)  → e₂·e₄ = +e₆
L5: (2,5,7)  → e₂·e₅ = +e₇
L6: (3,4,7)  → e₃·e₄ = +e₇
L7: (3,6,5)  → e₃·e₆ = +e₅
```

(Furey convention. **Do not change these.** Every downstream result depends on them.)

From these 7 directed triples, every other product follows by cyclicity and
antisymmetry. There are C(7,3) = 35 unordered triples of imaginary units.
Exactly 7 are collinear (associative). The remaining **28 are non-associative**.

## 4. The Graph Architecture

### 4.1 Nodes

```
Node = {
    id:          ℕ               -- unique node identifier
    state:       OctIdx          -- 0-indexed imaginary unit {0..6}
    tick_count:  ℕ               -- cumulative ticks to create this node
}
```

All nodes are imaginary-unit states. The real axis (e₀ = 1) is not a valid
node state — it represents a "real result" (blocked interaction, no successor).

### 4.2 Edges and Gluon Operators

Edges carry a **gluon operator**: an OctIdx in {0..6}.

**Gluon assignment (LOCKED, from `calc/gluon_assignment.py`):**

The gluons mediating exchange between Color i and Color j are exactly the
elements of Witt pair k, where {i,j,k} = {1,2,3}:

| Exchange | Gluon pair | Witt pair |
|----------|------------|-----------|
| Color 1 ↔ Color 2 | {e₃, e₄} = {2, 3} | Witt pair 3 (Color 3) |
| Color 1 ↔ Color 3 | {e₂, e₅} = {1, 4} | Witt pair 2 (Color 2) |
| Color 2 ↔ Color 3 | {e₁, e₆} = {0, 5} | Witt pair 1 (Color 1) |

This is the **Triality Rule**: the gluons for Colors i↔j live in the Witt pair
of the *third* color k. It is forced by Fano geometry alone — zero free parameters.

### 4.3 The Interaction Rule (Three-Body Product)

When node src (state `a`) emits gluon `g` to node dst (state `b`), compute
both bracketings of the triple product:

```
s_L = (e_a · e_g) · e_b   [left bracketing]
s_R =  e_a · (e_g · e_b)  [right bracketing]
```

**Alternativity Trigger:** The alternator [a,g,b] = (e_a·e_g)·e_b - e_a·(e_g·e_b)
vanishes whenever any two of a, g, b are equal (alternativity property). So:

- **Repeated element** (any two of a,g,b equal): `triggers(a,g,b) = False`
- **Fano collinear** ({a,g,b} is a Fano line): `triggers(a,g,b) = False`
- **Otherwise** (all distinct, off-line): `triggers(a,g,b) = True`

**Branching Protocol:**
- `triggers = False` (associative): s_L = s_R, cost **1 tick**, produce **1 successor**
  - If s_L is real (idx = -1): **blocked interaction**, no successor
- `triggers = True` (non-associative): s_L ≠ s_R, cost **2 ticks each**, produce **up to 2 successors**
  - One successor for s_L, one for s_R (if each has idx ≥ 0)

**Code:** `calc/mass_drag.py` functions `triggers()`, `three_body()`, `update_step()`

### 4.4 Mass = Tick Count

The inertial mass of a particle motif is proportional to its **minimum tick
count at first recurrence** of the initial state. This is the computational
drag — how many irreducible algebraic evaluations the universe must perform
to return the motif to its starting configuration.

Proved in Lean (`CausalGraphTheory/Mass.lean`):
- `tickMass_mono`: a node's mass equals its tick count
- `step_tickMass_nondecreasing`: mass never decreases

### 5.1 The Electron

The electron is a single node cycling through the **associative L1 subalgebra**
{e₁, e₂, e₃} (the quaternionic sub-algebra generated by L1):

```

```

Each hop: `fano_product(state, g)` computes the 2-body L1 product.
No Alternativity Trigger fires (all hops are within the associative L1 subalgebra).
Cost: 1 tick/hop.

**C_e = 3 ticks** (confirmed by simulation, no free parameters).

In code: `ELECTRON_CYCLE = [(0,1), (2,0), (1,2)]` (0-indexed state, gluon pairs).

The exchange schedule must be a **directed cyclic** exchange C1→C2→C3→C1
(each quark both emits and absorbs exactly once per 3-step period):


## 11. Known Failure Modes and Falsification History

**Do not repeat these mistakes:**

### F1: Trying to use ℝ or continuous mathematics
Every time you reach for π, a limit, or a probability, you are violating the
Prime Directive. Use integer counts, Fano line combinatorics, or Lean's
`Rat` instead.

### F2: The E8 Architecture
The E8 lattice of Cayley integers as node states was analyzed (RFC-008, now
deleted). Rejected because: Distler-Garibaldi theorem rules out E8 for
chiral fermions; the gate thresholds (θ_EM, θ_S, θ_W) are free parameters
(violating Prime Directive); the Albert algebra J₃(O) genesis has 27 free
parameters.

### F3: Asymmetric exchange schedule (MU-001 v1)
The proton exchange schedule `[(1,0), (1,2), (0,2)]` (in quark-index pairs)
made Color1 (Q[1]) perpetually a source and never a destination. Its state
never changed, making recurrence trivially dependent on only 2 of 3 quarks.
This gave C_p = 8, mu_COG = 2.667. Documented as falsification datum in
`claims/proton_electron_ratio.yml`. The correct schedule must be cyclic:
C1→C2→C3→C1.

### F4: SL(2,3) forcing B/A = √2 (KOIDE-001 retracted claim)
An earlier attempt claimed SL(2,3) ⊂ GL(3,2) forces cos²(θ) = 1/2, which
would give Q = 2/3. This was **wrong**. SL(2,3) forces equal phase spacing
(Z3 symmetry, 2π/3 between color planes) but does NOT pin the amplitude
ratio B/A. The calc/koide.py file was corrected and this claim retracted.

### F5: Editing without reading
The Edit tool requires reading the file first. On Windows with cp1252 encoding,
Unicode characters in the source (─, ✓, →) require running Python with `-X utf8`.
