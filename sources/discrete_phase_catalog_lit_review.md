# Discrete Phase Signatures Across Physics: Literature Review for COG

**Date:** 2026-02-26  
**Scope anchor:** `rfc/RFC-023_Discrete_Phase_Clocks_and_Relative_Phase_Interactions.md`  
**Focus question:** What independent physics clues suggest that subsystem dynamics can involve discrete phase classes?

---

## 1. Executive Summary

Across multiple domains, physics shows robust discrete phase structure:

1. **Spinor phase classes (`Z2`)** via `4pi` periodicity.
2. **Flux-phase periodicity** in Aharonov-Bohm systems (`h/e`, sometimes fractional harmonics).
3. **Phase-locking quantization** in Josephson/Shapiro phenomena (integer and fractional step families).
4. **Anyonic exchange phases** in 2D topological matter (fractional braid phase).
5. **Quantized geometric phases** (`Berry/Z_N`) in symmetry-protected topological settings.
6. **Gauge center sectors** (`Z_N`) in non-Abelian gauge theory via Polyakov-loop order structure.
7. **Temporal discrete phase order** (time crystals, multi-period response) in driven many-body systems.

These phenomena support the plausibility of COG's discrete-phase program, but none uniquely proves the specific `e7`/`Z4` mechanism.

## 2. Catalog of Clues

## 2.1 Spinor `4pi` periodicity (`Z2` phase class)

- Neutron interferometry confirms `4pi` periodicity under rotations for spinors.
- Physical meaning: two-valued phase class under `2pi` rotation (`+/-` sign), resolved only after `4pi`.

COG relevance: concrete precedent for intrinsic discrete phase classes tied to transformation cycles.

## 2.2 Aharonov-Bohm flux periodicity

- Mesoscopic normal-metal rings show conductance oscillations periodic in flux quantum.
- Interference phase is continuous in principle but operationally appears in quantized periodic sectors.

COG relevance: phase sectors modulate transport observables without requiring direct local force changes.

## 2.3 Josephson/Shapiro quantized phase locking

- AC Josephson relation produces quantized voltage steps under microwave drive.
- Fractional Shapiro families appear in richer junction dynamics.

COG relevance: interaction outcomes can lock to discrete phase/frequency ratios; supports phase-conditioned channel logic.

## 2.4 Anyonic exchange statistics

- Fractional quantum Hall anyons exhibit exchange phases different from bosons/fermions.
- Collision/interferometric experiments report extracted fractional statistical angles.

COG relevance: relative phase/statistics class can directly change correlation/interaction outputs.

## 2.5 Quantized Berry phases (`Z_N`)

- Topological/symmetry-protected systems can realize Berry phase quantized in `2pi/N`.
- `Z3`/`Z4` Berry invariants appear in model and numerical studies.

COG relevance: discrete phase indices can be symmetry-protected observables, not just numerical artifacts.

## 2.6 Center symmetry sectors in non-Abelian gauge theory

- Polyakov-loop framework organizes confinement/deconfinement via center (`Z_N`) structure.
- Distinct center sectors correspond to physically different thermal phases.

COG relevance: independent precedent for discrete sector labels tied to gauge-vacuum structure.

## 2.7 Multi-period temporal order (time crystals)

- Floquet systems show robust subharmonic response and, in some settings, multiple periods.
- Composite dynamics can yield richer cycle spectra than a base local drive period.

COG relevance: directly motivates testing period sets beyond 4 for composite motifs.

---

## 3. Claim Triage for COG

1. **Supported:** Discrete phase classes are common and physically consequential.
2. **Supported:** Relative phase can modulate interaction channels/correlations.
3. **Supported:** Composite systems can have longer/multiple periods than local primitives.
4. **Not supported:** Any direct theorem that COG must use exactly `e7`/`Z4`.
5. **Not supported:** Unknown phase entropy alone as an energy source.
