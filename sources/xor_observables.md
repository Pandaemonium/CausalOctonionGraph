# XOR Observables

Primary code: `calc/xor_observables.py`

## Purpose
1. Provide shared metrics for scenario and ensemble analysis.
2. Keep one consistent contract for entropy/transition/stability summaries.

## Included metrics
1. Base-8 decoding (`decode_base8_int`)
2. Shannon entropy from counts
3. Charge-sign flip count
4. Support-variation count
5. Pair-kind transition count
6. Scenario-level observables and ensemble aggregation
